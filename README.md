# MakerTech's Godot Voxel Terrain 
*This project is made for educational purposes*

In this project I explore different aspects of 3D development related to creating a voxel terrain in Godot.

You can watch the [related devlogs and tutorials here.](https://www.youtube.com/playlist?list=PLMQtM2GgbPEW8ElbOfc49QWV0eOquQiic)

Updates are exclusive for [Patreons](https://www.patreon.com/c/MakerTech) the first month after a new video is released - after this they are open sourced on GitHub.

## Topics covered so far:
- Creating the first Cubes
- 3D First Person Controller
- Locate Performance Issues Using Godot's Profiler
- GPU Instancing
- Procedural Mesh and ArrayMesh
- Intro to chunking and multi threading
- Greedy Meshing
- GDScript vs. C#
- Add / Remove blocks

## Next:
- Faster Greedy Meshing using Binary Meshing, Integers and bitshifts

  
### Controls:
- Movment: WASD / arrows
- Jump: Space
- Fly on/off: Enter
- Wireframe: F
- Exit: Escape
